### 任务目的
* 针对设计稿样式进行合理的HTML架构，包括以下但不限于：
```
*  掌握常用HTML标签的含义、用法
```
  * 能够基于设计稿来合理规划HTML文档结构
  * 理解语义化，合理地使用HTML标签来构建页面
  
* 掌握基本的CSS编码，包括以下但不限于：
```
*  了解CSS的定义、概念、发展简史
```
  * 掌握CSS选择器的含义和用法
  * 实践并掌握CSS的颜色、字体、背景、边框、盒模型、简单布局等样式的定义方式

### 任务描述
